# Support

Thanks for using Norse Architecture projects. Here is how to get help.

## How to Get Help

- **Documentation first.** Each repository's `README.md` and docs are the primary reference — please check them before opening anything.
- **GitHub Discussions** — for questions, ideas, and "how do I…" conversations, use the Discussions tab on the relevant repository when it is enabled.
- **GitHub Issues** — for confirmed bugs and feature requests. Use the issue templates where provided, and include versions, environment details, and a minimal reproduction.

## What Issues Are Not For

- General questions (use Discussions).
- Security vulnerabilities — see [SECURITY.md](SECURITY.md) for private reporting instructions. Never report security issues publicly.

## Response Expectations

These projects are maintained on a best-effort basis. Issues and discussions are triaged regularly, but there is no guaranteed response time. Clear, well-reproduced reports get attention fastest.

## Contributing

If you would like to fix the problem yourself, even better — see [CONTRIBUTING.md](CONTRIBUTING.md).
